# scrum-daily-standup-picker
This is a tool which can be used in Scrum teams to initiate the daily scrum meeting.

# Prerequisites
* Angular CLI (https://cli.angular.io/)

# Run
* Run "ng serve"
* Open browser and go to "http://localhost:4200"

# Features
* Plays random
